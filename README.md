# PricePulse AI Frontend

## Run locally

1. Install dependencies with `npm install` from the repository root or `npm install` inside `frontend`.
2. Start the app with `npm run dev`.
3. Set `VITE_API_BASE_URL` in `frontend/.env` if the backend is not running on `http://localhost:4000`.

## Deploy

- Build command: `npm run build`
- Output directory: `dist`
- Deploy to Vercel.
